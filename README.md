# HTML Analyzer

A partir de uma URL, obtem o trecho de texto contido no nível mais 
profundo da estrutura HTML de seu conteúdo.


# Pré-requisitos

- JDK 8 ou superior

# Execução

```java 
    javac HtmlAnalyzer.java
```

```java 
    java HtmlAnalyzer <URL>
```
